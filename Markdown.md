# 과제(8)

## 순환적 방법 피보나치

### 실행 시간

| 입력값(n) | 실행 시간 |
| --- | --- |
| 5 | 3.504초 |
| 10 | 3.16초 |
| 20 | 3.105초 |
| 33 | 3.374초 |
| 40 | 3.894초 |
| 45 | 3.333초 |

## 재귀적 방법 피보나치

### 실행 시간

| 입력값(n) | 실행 시간 |
| --- | --- |
| 5 | 4.107초 |
| 10 | 3.027초 |
| 20 | 3.425초 |
| 33 | 3.164초 |
| 40 | 3.768초 |
| 45 | 9.766초 |

### 결과

같은 값을 순환적 방법과 재귀적 방법 비교했을 때 입력값이 40까지는 차이가 없지만 입력값이 45부터는 실행 시간 차이가 많이 차이 나는 것을 알 수 있다.
